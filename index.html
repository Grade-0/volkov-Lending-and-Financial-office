Setup Snapshot

This build is ready for GitHub Pages plus shared Supabase storage once you add project credentials and run the supplied SQL.

1. Run the SQL in supabase-schema.sql inside the Supabase SQL Editor.
2. Copy supabase-config.example.js to supabase-config.js and paste your URL plus publishable key.
3. Invite staff accounts in Supabase Auth and sign in here from GitHub Pages.
4. Keep the JSON export and import tools for backup or migration workflows.
